# Amazon-Mobile-Insights-End-to-End-Data-Pipeline-Analytics
This project focuses on analyzing Amazon mobile product data to extract actionable insights regarding brand performance, pricing strategies, and customer satisfaction. The goal was to build an automated system that transforms raw, messy data into a professional decision-making dashboard.

# Tech Stack & Tools
Python (Pandas, SQLAlchemy): Used for advanced data cleaning, handling missing values, and automating the ETL process.
SQL Server: Served as the central data warehouse for storing cleaned and structured data.
Power BI: Used for data modeling, DAX calculations, and creating interactive visualizations.

# Key Accomplishments
## Automated ETL Pipeline:
Developed custom Python functions to professionally clean and validate data.
Resolved significant issues with missing and incorrect data (e.g., brand extraction from product names).
Automated the data migration from CSV to SQL Server.
## Advanced Data Modeling:
Used SQL to efficiently query and prepare datasets for visualization.
Implemented complex DAX measures in Power BI to calculate accurate average ratings and price distributions.
## Strategic Business Insights:
Identified the Top 15 and Bottom 15 brands based on customer sentiment.
Analyzed the correlation between Price and Ratings to help Amazon identify high-demand products and address quality issues in lower-rated ones

# Dashboard Features
Brand Performance Scorecard: View each brand's average price vs. average rating.
Deep-Dive Analysis: Granular view of individual products within each brand.
Trend Identification: Visual representation of how pricing impacts customer satisfaction.
